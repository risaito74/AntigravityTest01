## ※おまけメモ

すでにUnityプロジェクトが完成した状態で、GitHub Desktopでリポジトリを作る所からやる場合
- FIle > New Repository...を選択
- Git ignoreは「Unity」を選択（これをしないと数GB全部プッシュしようとして詰む）
- ローカルパスはUnityプロジェクトフォルダ名をぬいた状態「C:\unity\」にする。これにリポジトリ名（フォルダ名）が結合される
